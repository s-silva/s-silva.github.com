Fennec Media Suite

A Universal Media Player and Transcoder


2001-Present
http://fennec.sf.net
Source Code - GitHub

Fennec Media Project A complete suite for universal media playback/transcribing. The project is located at fennec.sf.net, released under GNU GPL (Open Source) license. Started working on the project around 2003-2004.

Have implemented features including - high definition audio/video playback and transcoding, ability to play and convert almost every audio/video file format available, visualizations, supports converting/audio CD ripping/joining, headphone preview, easy tag editing, file name formatting, channel separated equalizations, DSP effects, volume/gain control and equalization for conversion and so on... Runs on Microsoft Windows based systems. Programmed using C/C++ and x86 Assembly for FPU usage and for speed critical components.

Uses: C/C++/x86ASM/WinAPI/DirectX/OpenGL and various multimedia platforms
